To get started and for more information about Neo go to http://www.neo4j.org

Quick start: 

o add neo.jar and jta-spec1_0_1.jar to your classpath/project
o javadoc for Neo API can be found at http://api.neo4j.org 
  (in eclipse right click neo.jar Properties->Javadoc location)
o see javadoc for org.neo4j.api.core.EmbeddedNeo to get started
